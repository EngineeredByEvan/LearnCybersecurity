# Linux Essentials

## Introduction
Linux dominates servers and security tooling. Defenders need strong command-line fluency to inspect logs, processes, and configurations quickly.

## Worked Example
`cat /var/log/auth.log | grep Accepted`

## Exercise
Use `find` to locate `.log` files under `/var`.

## Hints
1. Use `find /var -name "*.log"`

## Solution
`find /var -name "*.log"`
