# Computing Fundamentals

## Introduction
Computers store data as bits. Binary (base 2) maps naturally to digital circuits. Hexadecimal (base 16) compresses binary for readability in security logs and memory analysis.

## Worked Example
- Binary `1101` = Decimal `13`
- Hex `0x2F` = Decimal `47`

## Exercise
Convert `0x3A` to decimal.

## Hints
1. Hex digits are powers of 16.
2. `3*16 + 10`.

## Solution
`58`
