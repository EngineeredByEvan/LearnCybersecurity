# SOC Operations Overview

## Introduction
SOC teams triage alerts, investigate indicators, and escalate incidents with clear documentation.

## Worked Example
Tier 1 validates alert fidelity, Tier 2 investigates scope, Tier 3 performs advanced analysis.

## Exercise
Write a short triage note with alert, evidence, and action taken.

## Hints
Include timestamp and host.

## Solution
Document alert source, affected asset, observed IOC, containment action.
